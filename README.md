# FeedbackKit SDK - Landing Page

Beautiful marketing website for the FeedbackKit SDK, an AI-powered Jira integration for iOS & Android mobile apps.

## Overview

This landing page showcases the FeedbackKit SDK, which brings AI-powered bug reporting directly into mobile applications. The SDK allows users to create Jira tickets with screenshots in seconds, built with SwiftUI & Jetpack Compose.

## Features Highlighted

- **AI-Powered Descriptions** - Automatically improve bug descriptions using ChatGPT
- **Screenshot Attachments** - Capture and attach screenshots to Jira tickets automatically
- **Beautiful Native Design** - Modern interface with dark mode support
- **Zero Backend Required** - Direct Jira API integration
- **Modern Architecture** - TCA for iOS, MVVM for Android
- **Secure & Private** - Data goes directly to your Jira instance
